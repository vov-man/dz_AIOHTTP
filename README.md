API, написанный на aiohttp(backend) для сайта объявлений.

Реализованы методы создания/удаления/редактирования объявления.

У объявления реализованны следующие поля:
заголовок
описание
дата создания
владелец

для запуска нужен .env фаил: 

DB_NAME=                           

DB_USER=                        

DB_PASSWORD=

DB_HOST=

DB_PORT=
